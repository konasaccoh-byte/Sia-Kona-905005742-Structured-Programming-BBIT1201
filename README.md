# Community Data Processing System

## Project Overview
The Community Data Processing System is a simple structured programming application developed using Python. The system is designed to help communities manage resident information and community issues efficiently.

This application replaces manual record-keeping methods with a computerized system that improves accuracy, organization, and accessibility of data.

---

## Objectives
The main objectives of the project are:

- To register community residents
- To store and manage resident information
- To record community issues and complaints
- To provide easy access to community records
- To demonstrate the use of structured programming concepts

---

## Features
The system includes the following features:

1. Register Resident
2. View Resident Records
3. Record Community Issues
4. View Recorded Issues
5. Exit the System

---

## Programming Concepts Used
This project applies structured programming principles such as:

- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling
